# Auto-Moderation Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add server-level auto-moderation (spam detection, duplicate blocking, word filtering, timeouts, kicks, and bans) that runs transparently on every message sent to a text channel.

**Architecture:** A synchronous `check_automod` helper is called inside `create_message` before the INSERT; if any rule fires, the message is rejected with HTTP 403 and the configured action (timeout/kick/ban) is applied to the sender. Spam detection runs after the INSERT and applies its action to the user going forward. All automod actions are recorded in `automod_logs`. The migration (`20240216000017_automod.sql`) is already in the repo and defines all five tables.

**Tech Stack:** Rust/Axum/sqlx (backend), React/Vitest (frontend). No new crates needed — all patterns follow existing handlers (see `handlers/audit.rs` for logger pattern, `handlers/bots.rs` for owner-check patterns).

//! Auto-moderation configuration API.
//!
//! Provides:
//! - `GET  /servers/:id/automod`       — Get config (owner only)
//! - `PATCH /servers/:id/automod`      — Upsert config (owner only)
//! - `GET  /servers/:id/automod/words` — List word filters (owner only)
//! - `POST /servers/:id/automod/words` — Add word filter (owner only)
//! - `DELETE /servers/:id/automod/words/:word` — Remove word (owner only)
//! - `GET  /servers/:id/automod/logs`  — List automod actions (owner only)
//! - `GET  /servers/:id/bans`          — List server bans (owner only)
//! - `DELETE /servers/:id/bans/:user_id` — Unban user (owner only)

use axum::{
    extract::{Path, State},
    http::StatusCode,
    Json,
};
use uuid::Uuid;

use super::shared::fetch_server;
use crate::{
    auth::AuthUser,
    error::{AppError, AppResult},
    models::{
        AddWordFilterRequest, AutomodConfig, AutomodLog, AutomodWordFilter,
        ServerBan, UpdateAutomodConfigRequest,
    },
    state::AppState,
};

const VALID_ACTIONS: &[&str] = &["delete", "timeout", "kick", "ban"];
const DEFAULT_AUTOMOD_LOG_LIMIT: i64 = 50;

// ============================================================================
// Config handlers
// ============================================================================

/// GET /servers/:id/automod — Fetch the server's automod config.
///
/// Returns 404 if no config has been created yet (owner hasn't configured anything).
pub async fn get_automod_config(
    Path(server_id): Path<Uuid>,
    State(state): State<AppState>,
    auth: AuthUser,
) -> AppResult<Json<AutomodConfig>> {
    let server = fetch_server(&state.pool, server_id).await?;
    if server.owner_id != auth.user_id() {
        return Err(AppError::Forbidden(
            "Only the server owner can view automod config".into(),
        ));
    }

    let config: Option<AutomodConfig> = sqlx::query_as(
        r#"
        SELECT server_id, enabled, spam_enabled, spam_max_messages, spam_window_secs,
               spam_action, duplicate_enabled, word_filter_enabled, word_filter_action,
               timeout_minutes, updated_at
        FROM automod_configs
        WHERE server_id = $1
        "#,
    )
    .bind(server_id)
    .fetch_optional(&state.pool)
    .await?;

    config.map(Json).ok_or_else(|| AppError::NotFound("No automod config found".into()))
}

/// PATCH /servers/:id/automod — Create or update the server's automod config.
///
/// Uses UPSERT — safe to call when no config exists yet.
pub async fn update_automod_config(
    Path(server_id): Path<Uuid>,
    State(state): State<AppState>,
    auth: AuthUser,
    Json(req): Json<UpdateAutomodConfigRequest>,
) -> AppResult<Json<AutomodConfig>> {
    let server = fetch_server(&state.pool, server_id).await?;
    if server.owner_id != auth.user_id() {
        return Err(AppError::Forbidden(
            "Only the server owner can update automod config".into(),
        ));
    }

    // Validate action values if provided.
    if let Some(ref action) = req.spam_action {
        if !VALID_ACTIONS.contains(&action.as_str()) {
            return Err(AppError::Validation(format!(
                "spam_action must be one of: {}",
                VALID_ACTIONS.join(", ")
            )));
        }
    }
    if let Some(ref action) = req.word_filter_action {
        if !VALID_ACTIONS.contains(&action.as_str()) {
            return Err(AppError::Validation(format!(
                "word_filter_action must be one of: {}",
                VALID_ACTIONS.join(", ")
            )));
        }
    }

    let config: AutomodConfig = sqlx::query_as(
        r#"
        INSERT INTO automod_configs (server_id)
        VALUES ($1)
        ON CONFLICT (server_id) DO UPDATE SET
            enabled             = COALESCE($2, automod_configs.enabled),
            spam_enabled        = COALESCE($3, automod_configs.spam_enabled),
            spam_max_messages   = COALESCE($4, automod_configs.spam_max_messages),
            spam_window_secs    = COALESCE($5, automod_configs.spam_window_secs),
            spam_action         = COALESCE($6, automod_configs.spam_action),
            duplicate_enabled   = COALESCE($7, automod_configs.duplicate_enabled),
            word_filter_enabled = COALESCE($8, automod_configs.word_filter_enabled),
            word_filter_action  = COALESCE($9, automod_configs.word_filter_action),
            timeout_minutes     = COALESCE($10, automod_configs.timeout_minutes),
            updated_at          = NOW()
        RETURNING server_id, enabled, spam_enabled, spam_max_messages, spam_window_secs,
                  spam_action, duplicate_enabled, word_filter_enabled, word_filter_action,
                  timeout_minutes, updated_at
        "#,
    )
    .bind(server_id)
    .bind(req.enabled)
    .bind(req.spam_enabled)
    .bind(req.spam_max_messages)
    .bind(req.spam_window_secs)
    .bind(req.spam_action)
    .bind(req.duplicate_enabled)
    .bind(req.word_filter_enabled)
    .bind(req.word_filter_action)
    .bind(req.timeout_minutes)
    .fetch_one(&state.pool)
    .await?;

    Ok(Json(config))
}

/// GET /servers/:id/automod/words — List all word filters for a server.
pub async fn list_word_filters(
    Path(server_id): Path<Uuid>,
    State(state): State<AppState>,
    auth: AuthUser,
) -> AppResult<Json<Vec<AutomodWordFilter>>> {
    let server = fetch_server(&state.pool, server_id).await?;
    if server.owner_id != auth.user_id() {
        return Err(AppError::Forbidden(
            "Only the server owner can view word filters".into(),

    let words: Vec<AutomodWordFilter> = sqlx::query_as(

    .bind(server_id)
    .fetch_all(&state.pool)
    .await?;

    Ok(Json(words))
}

/// POST /servers/:id/automod/words — Add a word to the blocklist.
pub async fn add_word_filter(
    Path(server_id): Path<Uuid>,
    State(state): State<AppState>,
    auth: AuthUser,
    Json(req): Json<AddWordFilterRequest>,
) -> AppResult<(StatusCode, Json<AutomodWordFilter>)> {
    let server = fetch_server(&state.pool, server_id).await?;
    if server.owner_id != auth.user_id() {
        return Err(AppError::Forbidden(
            "Only the server owner can add word filters".into(),
        ));
    }

    let word = req.word.trim().to_lowercase();
    if word.is_empty() {
        return Err(AppError::Validation("Word cannot be empty".into()));
    }

    let filter: AutomodWordFilter = sqlx::query_as(
        r#"
        INSERT INTO automod_word_filters (server_id, word, created_by)
        VALUES ($1, $2, $3)
        ON CONFLICT (server_id, word) DO UPDATE SET word = EXCLUDED.word
        RETURNING id, server_id, word, created_at
        "#,
    )
    .bind(server_id)
    .bind(&word)
    .bind(auth.user_id())
    .fetch_one(&state.pool)
    .await?;

    Ok((StatusCode::CREATED, Json(filter)))
}

/// DELETE /servers/:id/automod/words/:word — Remove a word from the blocklist.
pub async fn remove_word_filter(
    Path((server_id, word)): Path<(Uuid, String)>,
    State(state): State<AppState>,
    auth: AuthUser,
) -> AppResult<StatusCode> {
    let server = fetch_server(&state.pool, server_id).await?;
    if server.owner_id != auth.user_id() {
        return Err(AppError::Forbidden(
            "Only the server owner can remove word filters".into(),
        ));
    }

    sqlx::query(
        "DELETE FROM automod_word_filters WHERE server_id = $1 AND word = $2",
    )
    .bind(server_id)
    .bind(word.to_lowercase())
    .execute(&state.pool)
    .await?;

    Ok(StatusCode::NO_CONTENT)
}

## Chunk 2: Enforcement logic

### Task 3: Automod enforcement in `create_message`

**Files:**

- Modify: `server/src/handlers/automod.rs` (add `check_automod` public function)
- Modify: `server/src/handlers/messages.rs` (call `check_automod` before insert)
- Modify: `server/tests/automod_tests.rs` (add enforcement tests)

**Context:** `check_automod` is called in `create_message` **before** the INSERT for word-filter and duplicate checks. After the INSERT, spam is checked. If any pre-insert check fires with action `"delete"`, return `AppError::Forbidden("Message blocked by automod")`. For `"timeout"/"kick"/"ban"` actions, apply the action to the user and still return Forbidden (message is not saved).

Spam fires post-insert: the message is already saved, but the action (timeout/kick/ban) applies to future messages. For spam with action `"delete"`, the message is soft-deleted after insert.

The `check_automod` function signature:

```rust
pub async fn check_automod(
    pool: &PgPool,
    server_id: Uuid,
    channel_id: Uuid,
    user_id: Uuid,
    username: &str,
    content: &str,
    message_id: Option<Uuid>,  // None = pre-insert, Some = post-insert (spam check)
) -> AppResult<()>
```

Returns `Ok(())` if message is clean. Returns `Err(AppError::Forbidden(...))` if blocked.

// ============================================================================

/// Check automod rules for an incoming message.
///
/// - `message_id = None`: pre-insert call (word filter + duplicate + timeout check).
/// - `message_id = Some(id)`: post-insert call (spam check only).
///
/// Returns `Ok(())` if the message is allowed.
/// Returns `Err(AppError::Forbidden(...))` if the message is blocked.
pub async fn check_automod(
    pool: &PgPool,
    server_id: Uuid,
    channel_id: Uuid,
    user_id: Uuid,
    username: &str,
    content: &str,
    message_id: Option<Uuid>,
) -> AppResult<()> {
    // Fetch config — if none exists or automod is disabled, allow.
    let config: Option<AutomodConfig> = sqlx::query_as(
        r#"
        SELECT server_id, enabled, spam_enabled, spam_max_messages, spam_window_secs,
               spam_action, duplicate_enabled, word_filter_enabled, word_filter_action,
               timeout_minutes, updated_at
        FROM automod_configs WHERE server_id = $1
        "#,
    )
    .bind(server_id)
    .fetch_optional(pool)
    .await?;

    let config = match config {
        Some(c) if c.enabled => c,
        _ => return Ok(()),
    };

    if message_id.is_none() {
        // ── Pre-insert checks ────────────────────────────────────────────────

        // 1. Timeout check — blocked regardless of which rules fired it.
        let is_timed_out: bool = sqlx::query_scalar(
            "SELECT EXISTS(
                 SELECT 1 FROM automod_timeouts
                 WHERE user_id = $1 AND server_id = $2 AND expires_at > NOW()
             )",
        )
        .bind(user_id)
        .bind(server_id)
        .fetch_one(pool)
        .await?;

        if is_timed_out {
            return Err(AppError::Forbidden("You are currently timed out".into()));
        }

        // 2. Word filter check.
        if config.word_filter_enabled {
            let content_lower = content.to_lowercase();
            let words: Vec<String> = sqlx::query_scalar(
                "SELECT word FROM automod_word_filters WHERE server_id = $1",
            )
            .bind(server_id)
            .fetch_all(pool)
            .await?;

            if let Some(matched) = words.iter().find(|w| content_lower.contains(w.as_str())) {
                let matched = matched.clone();
                apply_action(
                    pool,
                    server_id,
                    channel_id,
                    user_id,
                    username,
                    &config.word_filter_action,
                    config.timeout_minutes,
                    "word_filter",
                    content,
                    Some(&matched),
                    None,
                )
                .await?;
                return Err(AppError::Forbidden("Message blocked by automod".into()));
            }
        }

        // 3. Duplicate check — always blocks (no configurable action).
        if config.duplicate_enabled {
            let is_duplicate: bool = sqlx::query_scalar(
                r#"
                SELECT EXISTS(
                    SELECT 1 FROM messages
                    WHERE author_id = $1
                      AND channel_id = $2
                      AND content = $3
                      AND deleted = FALSE
                      AND created_at > NOW() - INTERVAL '30 seconds'
                )
                "#,
            )
            .bind(user_id)
            .bind(channel_id)
            .bind(content)
            .fetch_one(pool)
            .await?;

            if is_duplicate {
                log_automod_action(
                    pool,
                    server_id,
                    Some(channel_id),
                    Some(user_id),
                    username,
                    "duplicate",
                    "delete",
                    Some(content),
                    None,
                )
                .await;
                return Err(AppError::Forbidden("Duplicate message blocked".into()));
            }
        }
    } else {
        // ── Post-insert check (spam) ─────────────────────────────────────────
        if config.spam_enabled {
            let count: i64 = sqlx::query_scalar(
                r#"
                SELECT COUNT(*) FROM messages
                WHERE author_id = $1
                  AND channel_id = $2
                  AND deleted = FALSE
                  AND created_at > NOW() - ($3::int || ' seconds')::interval
                "#,
            )
            .bind(user_id)
            .bind(channel_id)
            .bind(config.spam_window_secs)
            .fetch_one(pool)
            .await?;

            if count >= config.spam_max_messages as i64 {
                apply_action(
                    pool,
                    server_id,
                    channel_id,
                    user_id,
                    username,
                    &config.spam_action,
                    config.timeout_minutes,
                    "spam",
                    "",
                    None,
                    message_id,
                )
                .await?;
                // For spam with "delete" action, soft-delete the triggering message.
                if config.spam_action == "delete" {
                    if let Some(mid) = message_id {
                        let _ = sqlx::query(
                            "UPDATE messages SET deleted = TRUE WHERE id = $1",
                        )
                        .bind(mid)
                        .execute(pool)
                        .await;
                    }
                }
                // Spam doesn't return Forbidden for the current request (message already sent).
            }
        }
    }

    Ok(())
}

/// Apply an automod action to a user and log it.
async fn apply_action(
    pool: &PgPool,
    server_id: Uuid,
    channel_id: Uuid,
    user_id: Uuid,
    username: &str,
    action: &str,
    timeout_minutes: i32,
    rule_type: &str,
    content: &str,
    matched_term: Option<&str>,
    _message_id: Option<Uuid>,
) -> AppResult<()> {
    match action {
        "timeout" => {
            sqlx::query(
                r#"
                INSERT INTO automod_timeouts (user_id, server_id, expires_at, reason)
                VALUES ($1, $2, NOW() + ($3::int || ' minutes')::interval, $4)
                ON CONFLICT (user_id, server_id) DO UPDATE SET
                    expires_at = NOW() + ($3::int || ' minutes')::interval,
                    reason = $4
                "#,
            )
            .bind(user_id)
            .bind(server_id)
            .bind(timeout_minutes)
            .bind(format!("Automod: {}", rule_type))
            .execute(pool)
            .await?;
        }
        "kick" => {
            sqlx::query(
                "DELETE FROM server_members WHERE server_id = $1 AND user_id = $2",
            )
            .bind(server_id)
            .bind(user_id)
            .execute(pool)
            .await?;
        }
        "ban" => {
            sqlx::query(
                r#"
                INSERT INTO server_bans (user_id, server_id, reason)
                VALUES ($1, $2, $3)
                ON CONFLICT (user_id, server_id) DO NOTHING
                "#,
            )
            .bind(user_id)
            .bind(server_id)
            .bind(format!("Automod: {}", rule_type))
            .execute(pool)
            .await?;
            sqlx::query(
                "DELETE FROM server_members WHERE server_id = $1 AND user_id = $2",
            )
            .bind(server_id)
            .bind(user_id)
            .execute(pool)
            .await?;
        }
        _ => {} // "delete" — handled by caller (return Forbidden before insert)
    }

    log_automod_action(
        pool,
        server_id,
        Some(channel_id),
        Some(user_id),
        username,
        rule_type,
        action,
        Some(content),
        matched_term,
    )
    .await;

    Ok(())
}

/// Insert a row into `automod_logs`. Never fails loudly — logs errors and continues.
async fn log_automod_action(
    pool: &PgPool,
    server_id: Uuid,
    channel_id: Option<Uuid>,
    user_id: Option<Uuid>,
    username: &str,
    rule_type: &str,
    action_taken: &str,
    message_content: Option<&str>,
    matched_term: Option<&str>,
) {
    let result = sqlx::query(
        r#"
        INSERT INTO automod_logs
            (server_id, channel_id, user_id, username, rule_type, action_taken, message_content, matched_term)
        VALUES ($1, $2, $3, $4, $5, $6, $7, $8)
        "#,
    )
    .bind(server_id)
    .bind(channel_id)
    .bind(user_id)
    .bind(username)
    .bind(rule_type)
    .bind(action_taken)
    .bind(message_content)
    .bind(matched_term)
    .execute(pool)
    .await;

    if let Err(e) = result {
        tracing::error!(error = ?e, "Failed to write automod log");
    }
}

- [ ] **Step 3: Create `clients/web/src/components/servers/AutomodSettings.tsx`**

```tsx
import { useState, useEffect, type FormEvent } from "react";
import { api } from "../../api/client";
import type {
  AutomodConfig,
  AutomodWordFilter,
  AutomodLog,
  UpdateAutomodConfigRequest,
} from "../../types";
import styles from "./AutomodSettings.module.css";

type Tab = "rules" | "words" | "logs";

interface AutomodSettingsProps {
  serverId: string;
}

export function AutomodSettings({ serverId }: AutomodSettingsProps) {
  const [tab, setTab] = useState<Tab>("rules");
  const [config, setConfig] = useState<AutomodConfig | null>(null);
  const [words, setWords] = useState<AutomodWordFilter[]>([]);
  const [logs, setLogs] = useState<AutomodLog[]>([]);
  const [newWord, setNewWord] = useState("");
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState("");

  useEffect(() => {
    setLoading(true);
    api
      .getAutomodConfig(serverId)
      .then(setConfig)
      .catch(() => setConfig(null))
      .finally(() => setLoading(false));
    api
      .listWordFilters(serverId)
      .then(setWords)
      .catch(() => {});
    api
      .listAutomodLogs(serverId)
      .then(setLogs)
      .catch(() => {});
  }, [serverId]);

  async function patch(updates: UpdateAutomodConfigRequest) {
    setError("");
    try {
      const updated = await api.updateAutomodConfig(serverId, updates);
      setConfig(updated);
    } catch (e) {
      setError(e instanceof Error ? e.message : "Failed to update config");
    }
  }

  async function handleAddWord(e: FormEvent) {
    e.preventDefault();
    const word = newWord.trim();
    if (!word) return;
    try {
      const filter = await api.addWordFilter(serverId, word);
      setWords((prev) => [...prev, filter]);
      setNewWord("");
    } catch (e) {
      setError(e instanceof Error ? e.message : "Failed to add word");
    }
  }

  async function handleRemoveWord(word: string) {
    try {
      await api.removeWordFilter(serverId, word);
      setWords((prev) => prev.filter((w) => w.word !== word));
    } catch (e) {
      setError(e instanceof Error ? e.message : "Failed to remove word");
    }
  }

  if (loading) return <div className={styles.loading}>Loading…</div>;

  const c = config ?? {
    server_id: serverId,
    enabled: false,
    spam_enabled: false,
    spam_max_messages: 5,
    spam_window_secs: 5,
    spam_action: "delete" as const,
    duplicate_enabled: false,
    word_filter_enabled: false,
    word_filter_action: "delete" as const,
    timeout_minutes: 10,
    updated_at: "",
  };

  return (
    <div className={styles.container}>
      <h3 className={styles.heading}>Auto-Moderation</h3>

      {error && <div className={styles.error}>{error}</div>}

      <div className={styles.tabs}>
        <button
          className={tab === "rules" ? styles.activeTab : styles.tab}
          onClick={() => setTab("rules")}
        >
          Rules
        </button>
        <button
          className={tab === "words" ? styles.activeTab : styles.tab}
          onClick={() => setTab("words")}
        >
          Word Filter
        </button>
        <button
          className={tab === "logs" ? styles.activeTab : styles.tab}
          onClick={() => setTab("logs")}
        >
          Logs
        </button>
      </div>

      {tab === "rules" && (
        <div className={styles.rulesTab}>
          {/* Master toggle */}
          <label className={styles.toggle}>
            <input
              type="checkbox"
              aria-label="Enable Auto-Moderation"
              checked={c.enabled}
              onChange={(e) => patch({ enabled: e.target.checked })}
            />
            <span>Enable Auto-Moderation</span>
          </label>

          <fieldset className={styles.section} disabled={!c.enabled}>
            <legend className={styles.legend}>Spam Detection</legend>
            <label className={styles.toggle}>
              <input
                type="checkbox"
                checked={c.spam_enabled}
                onChange={(e) => patch({ spam_enabled: e.target.checked })}
              />
              <span>Enable spam detection</span>
            </label>
            {c.spam_enabled && (
              <div className={styles.subFields}>
                <label className={styles.field}>
                  Max messages
                  <input
                    type="number"
                    className={styles.numberInput}
                    min={1}
                    max={50}
                    value={c.spam_max_messages}
                    onChange={(e) =>
                      patch({ spam_max_messages: Number(e.target.value) })
                    }
                  />
                </label>
                <label className={styles.field}>
                  Per (seconds)
                  <input
                    type="number"
                    className={styles.numberInput}
                    min={1}
                    max={60}
                    value={c.spam_window_secs}
                    onChange={(e) =>
                      patch({ spam_window_secs: Number(e.target.value) })
                    }
                  />
                </label>
                <label className={styles.field}>
                  Action
                  <select
                    className={styles.select}
                    value={c.spam_action}
                    onChange={(e) =>
                      patch({
                        spam_action: e.target
                          .value as AutomodConfig["spam_action"],
                      })
                    }
                  >
                    <option value="delete">Delete message</option>
                    <option value="timeout">Timeout user</option>
                    <option value="kick">Kick user</option>
                    <option value="ban">Ban user</option>
                  </select>
                </label>
              </div>
            )}
          </fieldset>

          <fieldset className={styles.section} disabled={!c.enabled}>
            <legend className={styles.legend}>Duplicate Messages</legend>
            <label className={styles.toggle}>
              <input
                type="checkbox"
                checked={c.duplicate_enabled}
                onChange={(e) => patch({ duplicate_enabled: e.target.checked })}
              />
              <span>Block duplicate messages (within 30s)</span>
            </label>
          </fieldset>

          <fieldset className={styles.section} disabled={!c.enabled}>
            <legend className={styles.legend}>Word Filter</legend>
            <label className={styles.toggle}>
              <input
                type="checkbox"
                checked={c.word_filter_enabled}
                onChange={(e) =>
                  patch({ word_filter_enabled: e.target.checked })
                }
              />
              <span>Enable word filter</span>
            </label>
            {c.word_filter_enabled && (
              <label className={styles.field}>
                Action
                <select
                  className={styles.select}
                  value={c.word_filter_action}
                  onChange={(e) =>
                    patch({
                      word_filter_action: e.target
                        .value as AutomodConfig["word_filter_action"],
                    })
                  }
                >
                  <option value="delete">Delete message</option>
                  <option value="timeout">Timeout user</option>
                  <option value="kick">Kick user</option>
                  <option value="ban">Ban user</option>
                </select>
              </label>
            )}
            {(c.spam_action === "timeout" ||
              c.word_filter_action === "timeout") && (
              <label className={styles.field}>
                Timeout duration (minutes)
                <input
                  type="number"
                  className={styles.numberInput}
                  min={1}
                  max={10080}
                  value={c.timeout_minutes}
                  onChange={(e) =>
                    patch({ timeout_minutes: Number(e.target.value) })
                  }
                />
              </label>
            )}
          </fieldset>
        </div>
      )}

      {tab === "words" && (
        <div className={styles.wordsTab}>
          <form onSubmit={handleAddWord} className={styles.addWordForm}>
            <input
              className={styles.wordInput}
              placeholder="Add a word or phrase…"
              value={newWord}
              onChange={(e) => setNewWord(e.target.value)}
              maxLength={100}
            />
            <button type="submit" className={styles.addButton}>
              Add
            </button>
          </form>
          {words.length === 0 ? (
            <p className={styles.empty}>No words in filter.</p>
          ) : (
            <ul className={styles.wordList}>
              {words.map((w) => (
                <li key={w.id} className={styles.wordItem}>
                  <span className={styles.wordText}>{w.word}</span>
                  <button
                    className={styles.removeButton}
                    onClick={() => handleRemoveWord(w.word)}
                    aria-label={`Remove ${w.word}`}
                  >
                    ×
                  </button>
                </li>
              ))}
            </ul>
          )}
        </div>
      )}

      {tab === "logs" && (
        <div className={styles.logsTab}>
          {logs.length === 0 ? (
            <p className={styles.empty}>No automod actions yet.</p>
          ) : (
            <table className={styles.logsTable}>
              <thead>
                <tr>
                  <th>User</th>
                  <th>Rule</th>
                  <th>Action</th>
                  <th>Match</th>
                  <th>Time</th>
                </tr>
              </thead>
              <tbody>
                {logs.map((log) => (
                  <tr key={log.id}>
                    <td>{log.username}</td>
                    <td>{log.rule_type}</td>
                    <td>{log.action_taken}</td>
                    <td>{log.matched_term ?? "—"}</td>
                    <td>{new Date(log.created_at).toLocaleString()}</td>
                  </tr>
                ))}
              </tbody>
            </table>
          )}
        </div>
      )}
    </div>
  );
}
```
