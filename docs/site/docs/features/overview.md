# Features Overview

Together includes all the essentials for running a community — text channels, voice chat, DMs, roles, moderation, and more. This section covers each feature in detail.

## Communication

| Feature | Description |
|---------|-------------|
| [Text Channels & Threads](/features/channels) | Real-time chat with threading, pinning, and rich formatting |
| [Direct Messages](/features/direct-messages) | One-on-one DMs with full history |
| [Voice & Go Live](/features/voice-and-screen-share) | P2P WebRTC voice channels and screen sharing |

## Organization

| Feature | Description |
|---------|-------------|
| [Channel Categories](/features/channel-categories) | Group channels into collapsible sections |
| [Roles & Permissions](/features/roles-and-permissions) | Discord-compatible bitflag permission system |
| [Channel Permissions](/features/channel-permissions) | Fine-grained channel-level overrides |
| [Invites](/features/invites) | Invite links with configurable limits and expiry |

## Engagement

| Feature | Description |
|---------|-------------|
| [Reactions](/features/reactions) | Emoji reactions on any message |
| [Custom Emojis](/features/custom-emojis) | Upload and manage server emoji |
| [Polls & Events](/features/polls-and-events) | Built-in polling and server events |
| [Link Previews & GIFs](/features/link-previews-and-giphy) | URL unfurling and Giphy integration |
| [Presence Status](/features/presence-status) | Online, idle, DND, and custom status messages |

## Moderation

| Feature | Description |
|---------|-------------|
| [Auto-Moderation](/features/auto-moderation) | Spam filters, word filters, and rate limiting |
| [Member Moderation](/features/member-moderation) | Manual timeout, kick, ban, and mute actions |
| [Audit Logging](/features/audit-logging) | Complete audit trail of all moderation actions |
| [Message Search](/features/message-search) | Full-text search across all messages |
| [Message Editing & Deletion](/features/message-editing-deletion) | Full message lifecycle management |
| [Message Pinning](/features/message-pinning) | Pin important messages for easy access |

## Platform

| Feature | Description |
|---------|-------------|
| [Authentication](/features/authentication) | JWT-based auth with bcrypt password hashing |
| [Bot API](/reference/bot-api) | Build bots with token-based authentication |
| [Webhooks](/reference/webhooks) | HMAC-SHA256 signed webhook integrations |
